## ğŸ¯ Customization Examples

### **Change Primary Color**
Edit `frontend/tailwind.config.js`:
```javascript
colors: {
  secondary: '#FF6B6B',  // Change to your color
}
```

### **Increase Animation Speed**
Edit `frontend/tailwind.config.js`:
```javascript
animation: {
  slideUp: 'slideUp 0.4s ease-out',  // Faster (0.4s instead of 0.6s)
}
```

### **Add New Animation**
1. Add keyframes in `frontend/src/index.css`
2. Register in `tailwind.config.js`
3. Use as `animate-yourAnimationName`
